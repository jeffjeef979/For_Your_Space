# Features Overview

EventFlow combines 10 core modules into a single-screen unified dashboard. Each module is accessible via bottom navigation (mobile) or tab bar (desktop) without full-page navigations.

## Module Map

| Module | Tab | Description |
|---|---|---|
| Interactive Map | Map | Full-screen Google Maps with venue marker, geolocation, and routing |
| Street View Approach | Map | Panoramic imagery synced to user's position as they walk to venue |
| Event Feed | Events | Luma-inspired cards showing sessions, speakers, and schedule conflicts |
| Voice Assistant | AI | Tap-to-speak with real-time transcription and AI-powered responses |
| Digital Name Cards | Network | LinkedIn-style profile cards for all participants |
| Commute Planner | Commute | Travel time between events with driving and transit options |
| Smart Suggestions | For You | AI-recommended people and events based on user's background |
| Check-In Flow | Check-In | Sequential reveal of WiFi, table, and timeline after arrival |
| WhatsApp Connector | AI/Chat | Messages forwarded to organizer's WhatsApp in real time |
| Organizer Dashboard | (Role) | Live attendance, request queue, and analytics |

## Attendee Journey

The attendee experience follows a natural flow from arrival to engagement:

1. **Pre-arrival** — Map shows venue location, commute planner estimates travel time between sessions
2. **Approaching** — Geolocation triggers wayfinding, Street View shows what to look for
3. **Arrival** — Check-in confirms presence, reveals WiFi password and table number
4. **During event** — Voice assistant answers questions, event feed shows schedule, networking panel connects attendees
5. **Between sessions** — Smart suggestions recommend next sessions and people to meet

## Organizer Journey

The organizer has a separate dashboard view with real-time operational data:

1. **Pre-event** — Configure WiFi password, table assignments, event timeline
2. **During event** — Monitor live attendance map, see who has checked in
3. **Requests** — Receive and respond to attendee queries via WhatsApp
4. **Post-event** — Review analytics, attendance data, and engagement metrics
